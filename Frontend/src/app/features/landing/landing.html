What DevHub gives you

A focused set of tools built around real developer workflows.

Public & Private Repositories

Create repositories and choose who can see them. Private repos are protected — any unauthorized request returns 403 Forbidden. You own your code.

Push with devhub CLI

Use the devhub push command to send your code to a repository. Familiar Git-like workflow, no setup overhead.

File Tree Browser

Navigate your repository just like GitHub — browse folders and files, click into any file to see its contents. Built on a recursive tree structure in the backend.

Commit History

Every save creates a commit — message, author, and timestamp. View your full project timeline and see exactly who changed what and when.

Issues & Collaboration

Anyone can open an issue on a public repo and leave comments. Only the repository owner can close or delete it — role-based access built into every action.

Access Control (RBAC)

Ownership is enforced at the API level. Visitors browse, contributors comment, owners manage. Permissions are checked on every request — not just in the UI.
